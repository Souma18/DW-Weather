# DW-Weather
Dự án này xây dựng một pipeline tự động để:  Thu thập dữ liệu từ nhiều nguồn (API, HTML, JSON, v.v.).  Xử lý, làm sạch và chuẩn hoá dữ liệu bằng Python.  Tải dữ liệu vào Data Warehouse (BigQuery/Redshift/Snowflake/Postgres tuỳ setup).  Quản lý toàn bộ luồng công việc bằng Apache Airflow.
